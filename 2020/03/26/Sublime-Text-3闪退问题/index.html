Sublime Text 3闪退问题

发表于 2020-03-26

(ÒωÓ๑ゝ∠)_

1.关闭Sublime Text 3

2.根据不同的系统，进入如下对应目录

  • OS X: ~/Library/Application Support/Sublime Text 3
  • Windows: %APPDATA%\Sublime Text 3
  • Linux: ~/.config/sublime-text-3

3.删除设置文件

del：Local/Session.sublime_session

4.重启Sublime Text 3

5.注意事项

1.重新启动后，会重新创建设置文件，此时软件将回到原始状态。

2.在OS X上，Library文件夹在默认情况下是隐藏的。请在Finder中键入~/Library

（Go -> Go to Folder）

参考自sublimetext官方文档
